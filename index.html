Hybrid Code Quality Analysis: Automatic Formatters + Optional AI Corrections

Introduction

The AI Code Review Bot is a revolutionary hybrid system that combines the speed of automatic code formatters with the intelligence of AI-powered corrections. It works locally on your machine and creates pull requests with fixed code, providing the best of both worlds: fast automatic fixes for common issues + intelligent AI corrections for complex problems.

Key Features

  • Hybrid Approach: Automatic formatters (autopep8, isort) + Optional AI corrections
  • Local Operation: Runs entirely on your machine, no GitHub Actions needed
  • No API Required: Works with just automatic formatters for basic fixes
  • Optional AI Enhancement: Add OpenAI API key for advanced code corrections
  • PR Creation: Automatically creates pull requests for easy code review
  • Industry Standards: Enforces PEP 8 style guidelines and best practices
  • Multi-Tool Integration: Uses pylint, flake8, autopep8, and isort

How It Works

  1. Developer runs the bot locally: python .github/scripts/review_pr.py
  2. The bot scans all Python files in the current directory for linter issues
  3. First: Applies automatic formatting fixes using autopep8 and isort (fast, no API needed)
  4. Then: If OpenAI API key is provided, uses AI to fix remaining complex issues
  5. Creates a new git branch with the fixes
  6. Commits changes and pushes to GitHub
  7. Automatically creates a pull request for code review
  8. Merge the PR to apply the corrections to your codebase

System Architecture

Workflow Components

  • .github/scripts/review_pr.py: Main Python script with hybrid approach
  • .github/scripts/requirements.txt: Python dependencies (openai, pylint, flake8, autopep8, isort)
  • Automatic Formatters: autopep8 (PEP8 formatting), isort (import sorting)
  • Static Analysis Tools: Pylint for code quality, Flake8 for style enforcement
  • AI Enhancement: OpenAI GPT-3.5 for complex code corrections (optional)
  • Git Integration: Automatic branch creation, commit, and PR creation via GitHub CLI

Setup Instructions

  1. Fork or clone the repository
  2. Install dependencies: pip install -r .github/scripts/requirements.txt
  3. Authenticate GitHub CLI: gh auth login
  4. Optional: Set OpenAI API key for AI corrections: set OPENAI_API_KEY=your_key_here
  5. Run the bot locally: python .github/scripts/review_pr.py
  6. The bot will scan files, apply fixes, and create a pull request automatically
